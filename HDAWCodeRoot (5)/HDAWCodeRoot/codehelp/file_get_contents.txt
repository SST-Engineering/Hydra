<h2>file_get_contents</h2>
<i><b>Fetch contents of file</b></i><br/>
$result = file_get_contents(<i>file_name</i>)<br/>
<i>Return contents as string, false on failure</i><br/>
Fetch contents as a string from a file in the working directory
<br/>
<i>Example</i><br/>
$a = file_get_contents("my_data.csv");
