<h2>make_directory</h2>
<i><b>Make a new directory</b></i><br/>
$result = make_directory(<i>directory_name</i>)<br/>
<i>Make false on failure, true if made or if it already exists (not an error)</i><br/>
Make a new directory, in not exists, in the working directory
<br/>
<i>Example</i><br/>
$a = make_directory("MyWorkspace");
