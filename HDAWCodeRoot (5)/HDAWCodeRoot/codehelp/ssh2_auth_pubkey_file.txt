<h2>ssh2_auth_pubkey_file</h2>
<i><b>Authenticate using a public key file</b></i><br/>
$result = ssh2_auth_pubkey_file(username,pubfilefilename,privkeyfile,passphrase);<br/>
<i>Returns true, on success, false on fail </i><br/>

<br/>
<i>Example</i><br/>
ssh2_auth_pubkey_file("tjones","/home/srolph/fluentkey.pub","/home/srolph/fluentkey","hello world");<br/>
